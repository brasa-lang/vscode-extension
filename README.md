# Brasa Support for Visual Studio Code

This extension provides support for the [Brasa Programming Language](https://brasa-lang.github.io/). It provides **Syntax Highlighting** and
**Code Snippets** for common structures.

## How to use

Create a new file wit the `.brasa` extension (e.g, `main.brasa`). VS Code will automatically recognize the language and activate the extension.
